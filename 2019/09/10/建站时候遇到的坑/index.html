突发奇想

暑假的时候恶补了一下Html、css等的前端知识，又学习了Python的Web框架Django，突发奇想自己想建个站来放自己喜欢的东西上去。

配置环境

  • 语言环境：Python3.7.3（win上）,Python3.6.5(服务器上)
  • 所需安装环境：
    • 前端：semantic、highchart、masonry（以后再补）
    • 后端：Django、mongoengine、nginx、uwgsi
    • 数据库：mongodb4.0.3
    • 系统：centos6

开始安装

win10上环境安装不再说明,且主要说明后端环境配置

重要！！！安装之前先把要使用的端口开放

Centos6：
iptables -I INPUT -p tcp --dport 80 -j ACCEPT    （开放80端口）
netstat -lntup   查看所有tcp端口监听
ps -ef | grep nginx  查看nginx进程
netstat -unltp|grep nginx  查看nginx端口监听
/usr/local/nginx/sbin/./nginx -s reload

mongodb

1.下载MongoDB

官网下载地址 https://www.mongodb.com/download-center#community

下载地址 32位 http://dl.mongodb.org/dl/linux/i686

wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-rhel62-4.0.1.tgz

2.解压并复制到/usr/local/mongodb

tar -xvf mongodb-linux-x86_64-rhel62-4.0.1.tgz

3.创建数据库存放目录和日志存放文件

mkdir -p /data/db
touch /data/db/dblogs

4.启动MongoDB

/usr/local/mongodb/bin/mongod --dbpath=/data/db --logpath=/data/db/dblogs --logappend --fork

5.数据库的备份及恢复

 windows:
//备份
mongodump -h 127.0.0.1:27017 -d test -o c:\data\dump
//恢复,test是数据库名
 mongorestore -h 127.0.0.1:27017 -d test --directoryperdb c:\data\dump\test
linux:
//恢复
/usr/local/mongo/bin/mongorestore -d db名 目录
